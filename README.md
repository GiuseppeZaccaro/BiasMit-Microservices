# BiasMit-Microservices
Il progetto ingegnerizza la mitigazione dei bias negli LLM tramite Activation Steering, trasponendo la ricerca in un’architettura a microservizi con Docker Compose. Un orchestratore Spring Boot e un servizio Python (FastAPI) gestiscono i risultati di Mistral, Llama 3 e Qwen tramite Docker Volumes, simulando l'inferenza per limiti hardware.
